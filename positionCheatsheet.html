Name Description Code Example
Static Static position is the default for elements, and doesn't give elements any special positioning other than to fit the flow of the page. div.static {
position: static;
border: 3px solid #73AD21;
}
Relative You can position the element relative to it's default static position using offset values of up, down, left and right. .green-box {
background-color: green;
position: relative;
top: 50px;
left: 120px;
}
Absolute Other elements on a page will ignore absolute elements so you are free to move it wherever you wish. main{
position: absolute;
top: 115px;
left:0px;
right:0px;
}
Fixed If you set an elements position to fixed, it will not scroll with the page but will remain in it's place. This is commonly done for headers and footers. .title {
position: fixed;
top: 0px;
left: 0px;
}
Sticky A sticky element will scroll with the page until a specified point, defined by offset values, at which it will stick. .box-bottom {
background-color: darkgreen;
position: sticky;
top: 240px;
}
